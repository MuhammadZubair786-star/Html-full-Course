Basic Structure in Html

  1. DOCTYPE Declaration

    Explanation: Yeh declaration browser ko batata hai ki document HTML5 mein likha gaya hai. Yeh HTML document ka version specify karta hai.

  2. HTML Tag

    Explanation: Yeh root element hai jo saare HTML content ko wrap karta hai. lang attribute browser ko document ki language batata hai, is case mein English.

  3. Meta Charset Tag

    Explanation: Yeh tag document ka character encoding specify karta hai. UTF-8 widely used encoding hai jo almost saare characters ko support karti hai.

  4. Meta Viewport Tag

    Explanation: Yeh tag responsive web design ke liye important hai. Yeh tag browser ko batata hai ki page ko device ke width ke according scale karna hai, isse page mobile devices par properly display hota hai.

  5. Title Tag

    Explanation: Yeh tag browser tab mein display hone wala page title set karta hai. Yeh SEO aur usability ke liye bhi important hai.

  6. Body Tag

    Explanation: Yeh section actual content contain karta hai jo browser mein display hota hai, jaise headings, paragraphs, lists, images, etc.
